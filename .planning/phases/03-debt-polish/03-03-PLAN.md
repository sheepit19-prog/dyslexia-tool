**Add a sensitivity threshold calculator:**

```typescript
interface SensitivityThresholds {
  backspaceThreshold: number
  pauseThreshold: number
  snoozeDuration: number
}

async function getSensitivityThresholds(): Promise<SensitivityThresholds> {
  // Default sensitivity = 5 (medium)
  const result = await chrome.storage.local.get('settings')
  const sensitivity = result.settings?.companionSensitivity ?? 5

  // Scale thresholds inversely with sensitivity:
  // Sensitivity 1 (lenient): more backspaces needed, longer pause, longer snooze
  // Sensitivity 10 (aggressive): fewer backspaces, shorter pause, shorter snooze
  
  const factor = sensitivity / 5  // 0.2 at sensitivity 1, 2.0 at sensitivity 10
  
  return {
    backspaceThreshold: Math.max(1, Math.round(BACKSPACE_THRESHOLD / factor)),
    pauseThreshold: Math.round(PAUSE_THRESHOLD / factor),
    snoozeDuration: Math.round(SNOOZE_DURATION / factor),
  }
}
```

**Update detection logic to use dynamic thresholds:**

In the `keydownHandler` function, replace:
- `backspaceCount >= BACKSPACE_THRESHOLD` → use `await getSensitivityThresholds()` and compare against `thresholds.backspaceThreshold`
- `timeSinceLastOffer > SNOOZE_DURATION` → compare against `thresholds.snoozeDuration`

In the `pauseCheckInterval` callback, replace:
- `timeSinceTyping > PAUSE_THRESHOLD` → use dynamic `thresholds.pauseThreshold`
- `timeSinceLastOffer > SNOOZE_DURATION` → use dynamic `thresholds.snoozeDuration`

**Performance consideration:** Cache the sensitivity setting in companionState and refresh it periodically (e.g., every 60 seconds) rather than reading chrome.storage.local on every keystroke. Read once on module init, refresh in the pauseCheckInterval that already runs every 500ms.

```typescript
// Add to companion state refresh
let cachedThresholds: SensitivityThresholds | null = null
let thresholdsLoadedAt = 0

async function getThresholds(): Promise<SensitivityThresholds> {
  const now = Date.now()
  if (cachedThresholds && now - thresholdsLoadedAt < 60000) {
    return cachedThresholds
  }
  cachedThresholds = await getSensitivityThresholds()
  thresholdsLoadedAt = now
  return cachedThresholds
}
```

This ensures threshold reads are infrequent (once per minute max) while still allowing the user to change sensitivity and see results within a minute.

CRITICAL: The default (sensitivity=5) must produce the SAME behavior as the current hard-coded constants (BACKSPACE_THRESHOLD=3, PAUSE_THRESHOLD=10000, SNOOZE_DURATION=300000). Factor = 5/5 = 1.0, so division by 1.0 gives identical values. ✓

**Step 2: Create a safe style injection helper** (add to both UI modules or create a shared utility)

```typescript
function applyStylesSafe(element: HTMLElement, styles: Record<string, string>): boolean {
  try {
    const cssText = Object.entries(styles).map(([k, v]) => `${k}: ${v}`).join('; ')
    element.style.cssText = cssText
    // Verify styles were applied (CSP may silently block)
    return element.style.cssText.length > 0
  } catch {
    return false  // CSP blocked
  }
}
```
