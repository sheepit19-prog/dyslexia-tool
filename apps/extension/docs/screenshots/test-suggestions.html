📝 Spelling Suggestions Test

Use this page to test and capture screenshots of the Dyslexia Tool spelling suggestions feature.

📋 How to Test:

  1. Enable Companion Mode - Click the Dyslexia Tool icon and make sure "Companion Mode" is ON (blue toggle)
  2. Type a misspelled word - Use one from the list below
  3. Press Backspace 3+ times - The companion notification will appear
  4. Click "Show suggestions" - The suggestions panel will appear
  5. Capture screenshot! - Press F12 → Ctrl+Shift+P → "screenshot"

📚 Words to Test (These have suggestions):

becuase → because recieve → receive beleive → believe seperate → separate definately → definitely thier → their, there freind → friend taht → that jsut → just hte → the btu → but adn → and
💡 Tip: Start typing a sentence like "I did this becuase" then press backspace repeatedly on the misspelled word to trigger the companion.
