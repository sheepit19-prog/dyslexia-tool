# Screenshot Capture Guide

## Chrome Web Store Screenshot Requirements

| Requirement | Specification |
|-------------|---------------|
| **Minimum** | 1 screenshot |
| **Maximum** | 5 screenshots |
| **Recommended** | 3-4 screenshots |
| **Resolution** | 1280x800 or 640x400 minimum |
| **Format** | PNG or JPEG |
| **File Size** | Max 10MB each |

---

## Required Screenshots

### Screenshot 1: Extension Popup (Main Interface)
**What to capture:** The full extension popup showing all features

**Steps:**
1. Open any webpage (e.g., google.com)
2. Click the Dyslexia Tool icon in the toolbar
3. Ensure popup is fully visible
4. Capture: Full popup showing all toggles (Font, Read Aloud, Reading Ruler, Companion, Voice Note)

**Annotations to add:**
- Arrow pointing to "Companion Mode" toggle
- Callout: "Smart assistance when you need it"

**Dimensions:** 400x600 (popup native size) or scaled to 1280x800

---

### Screenshot 2: Companion in Action
**What to capture:** The companion notification appearing while typing

**Steps:**
1. Open a text editor (Google Docs, Gmail compose, etc.)
2. Type a few words, then press Backspace 3+ times
3. Wait for the companion notification to appear
4. Capture: The notification showing "Need help with spelling?"

**Annotations to add:**
- Circle around the notification
- Callout: "Gentle suggestions, never intrusive"
- Arrow showing "Show suggestions" button

**Dimensions:** 1280x800

---

### Screenshot 3: Spelling Suggestions Panel
**What to capture:** The suggestions panel showing correction options

**Steps:**
1. In a text field, type a misspelled word (e.g., "becuase")
2. Trigger companion (backspace 3x) or use popup
3. Click "Show suggestions"
4. Capture: The suggestion panel with correction options

**Annotations to add:**
- Highlight the suggestions
- Callout: "Common dyslexic misspellings corrected"
- Show the replacement in action

**Dimensions:** 1280x800

---

### Screenshot 4: Reading Features
**What to capture:** Reading ruler and text-to-speech in use

**Steps:**
1. Open a news article or long-form content
2. Enable Reading Ruler from popup
3. Select some text
4. Show the blue highlight line across the text

**Annotations to add:**
- Arrow pointing to reading ruler line
- Callout: "Stay focused while reading"
- Show TTS button highlighted

**Dimensions:** 1280x800

## Screenshot Capture Methods

### Method 1: Chrome DevTools (Recommended)

1. **Open DevTools:**
   - Press `F12` or `Ctrl+Shift+I` (Windows) / `Cmd+Opt+I` (Mac)

2. **Toggle Device Toolbar:**
   - Click the device icon or press `Ctrl+Shift+M`

3. **Set Dimensions:**
   - Enter `1280` x `800` in the dimension fields

4. **Capture:**
   - Press `Ctrl+Shift+P` (Windows) / `Cmd+Shift+P` (Mac)
   - Type "screenshot"
   - Select "Capture full size screenshot"

### Method 2: Screenshot Extensions

Install a screenshot extension:
- **Nimbus Screenshot** - Full page capture
- **Awesome Screenshot** - Annotation tools included
- **Lightshot** - Quick captures

### Method 3: Native OS Tools

**Windows:**
- `Win+Shift+S` for snipping tool
- `PrtScn` for full screen

**Mac:**
- `Cmd+Shift+4` for selection
- `Cmd+Shift+5` for full controls

## Editing Screenshots

### Recommended Tools

**Free:**
- **Canva** - Templates, annotations, professional look
- **GIMP** - Full-featured image editor
- **Paint.NET** - Simple Windows editor

**Paid:**
- **Photoshop** - Professional editing
- **Figma** - Great for UI annotations

### What to Add

1. **Device Frame** (optional):
   - Put screenshots in a browser frame for context
   - Makes them look more professional

2. **Callouts & Arrows:**
   - Highlight key features
   - Use consistent colors (match your brand)

3. **Text Overlays:**
   - Brief feature descriptions
   - Keep text minimal and readable

4. **Numbering:**
   - If showing a flow, number the steps
   - "1. Type", "2. Get suggestions", "3. Replace"

## Screenshot Checklist

Before uploading, verify:

- [ ] All screenshots are at least 1280x800 pixels
- [ ] Text is readable (not too small)
- [ ] No sensitive/personal information visible
- [ ] Consistent style across all screenshots
- [ ] Features are clearly highlighted
- [ ] File names are descriptive:
  - `screenshot-1-popup.png`
  - `screenshot-2-companion.png`
  - `screenshot-3-suggestions.png`
  - `screenshot-4-reading.png`

## After Capture

1. **Save files** to `apps/extension/docs/screenshots/`
2. **Name them** in the order they should appear
3. **Upload** during Chrome Web Store submission
4. **Preview** how they look in the store listing

---

## Tips for Better Screenshots

✅ **DO:**
- Use clean, uncluttered webpages
- Show the extension solving a problem
- Include callouts highlighting key features
- Keep text overlays short and clear
- Use high contrast for visibility

❌ **DON'T:**
- Show personal information (emails, names)
- Use blurry or low-resolution images
- Overcrowd with too many annotations
- Mix different visual styles
- Show error states or loading screens
